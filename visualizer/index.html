Using the Full PSIPro .ino Block Export

1. Create and save one or more frames.
2. Select All saved frames: Full PSIPro .ino block from the export dropdown.
3. Set a PSIPro command number.
4. Set the frame delay. Lower = faster animation.
5. Optional: set sequence runtime. Example: 5000 runs about 5 seconds before PSIPro resumes normal/default behavior. Use 0 for normal PSIPro runtime handling.
6. Copy each generated code block separately:
  • Frame Data / PROGMEM Patterns → paste near your other PSIPro letter/pattern arrays.
  • Playback Function → paste near your other PSIPro pattern functions.
  • Command Handler Case → paste inside the existing PSIPro command switch near the other case statements.
  • PSIPro Trigger Command → use with MarcDuino, DroidLink, serial monitor, or testing.
7. Search your PSIPro file for existing case statements, switch, or displayMatrixColor( to find the correct command handler section.
